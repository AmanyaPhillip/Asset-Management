# Contributing

Thanks for your interest in improving this project.

## Setup

1. `npm install` (in the `WebPortal/web-portal` directory)
2. Copy `.env.example` to `.env.local` and add your Supabase / Stripe keys
3. `npm run dev`

## Before opening a PR

- `npm run lint` and `npm run build` pass.
- Keep commits small with clear, present-tense messages.

## Reporting issues

Open a GitHub issue with steps to reproduce.
